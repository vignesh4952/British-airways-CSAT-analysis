# British Airways CSAT Analysis

This project focuses on analyzing **British Airways customer satisfaction (CSAT)**
using customer review data to understand passenger experience and service quality.

---

## Project Objective
- Analyze customer satisfaction levels
- Identify key factors affecting passenger experience
- Understand trends in ratings and reviews
- Present insights through dashboards and reports

---

## Tools & Technologies
- Power BI
- Microsoft Excel
- Data Analysis & Visualization

---

## Files in This Repository
- **British Air Way Dashboard.pbix**  
  Power BI dashboard showing customer satisfaction insights

- **British Airways CSAT Analysis.docx**  
  Detailed project report explaining analysis and findings

---

## Key Insights
- Overall customer satisfaction trends
- Areas of strong and weak passenger experience
- Actionable insights to improve service quality

---

## Outcome
This analysis helps in understanding customer sentiment and provides insights
that can support data-driven decisions to improve airline services.
